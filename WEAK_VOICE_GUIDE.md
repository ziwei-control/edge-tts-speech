## 🌸 柔弱版特色

### 声音特征

| 特征 | 实现方式 | 效果 |
|------|---------|------|
| **喘气声** | 添加"呼～"、"呼..." | 模拟呼吸声 |
| **延迟感** | 用"..."表示停顿 | 断断续续 |
| **轻轻柔柔** | 语速 -20%，音量 -10% | 更慢更轻 |
| **虚弱语气** | "好累"、"别走"、"陪我" | 示弱词汇 |
| **拖长音** | 用"～"表示 | 虚弱拖音 |

## 🔧 技术实现

### 文案标记

| 标记 | 含义 | 示例 |
|------|------|------|
| **呼～** | 深喘气 | 句首 |
| **呼...** | 浅喘气 | 句中 |
| **...** | 停顿/延迟 | 词之间 |
| **～** | 拖长音 | 句尾 |

### 参数配置

```python
# 柔弱版配置
{
    "rate": -20,      # 语速放慢 20%
    "volume": -10,    # 音量降低 10%
    "voice": "xiaoxiao"  # 温柔女声
}
```

### 生成脚本

```python
# generate_weak.py
WEAK_TEXTS = {
    "欢迎": "呼～欢迎...宝宝...来看我...呼...好累哦...💕",
    "感谢": "呼～谢谢...宝宝...的关注...呼...好开心...❤️",
}

WEAK_VOICES = {
    "xiaoxiao": {"rate": "-20%", "volume": "-10%"},
    "xiaoxuan": {"rate": "-20%", "volume": "-10%"},
    "xiaoyi": {"rate": "-15%", "volume": "-5%"},
}
```
