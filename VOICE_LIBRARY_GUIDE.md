### 数据加载

**自动扫描目录：**
```javascript
// 读取 output/demo/, output/natural/ 等目录
fetch(`output/${version}/`)

// 解析 MP3 文件名
xiaoxiao_通用.mp3 → {
    voiceName: 'xiaoxiao',
    scene: '通用',
    path: 'output/demo/xiaoxiao_通用.mp3'

```

---

### 搜索过滤

**三重匹配：**
```javascript
filteredVoices = voices.filter(v => 
    v.voiceName.includes(search) ||   // 语音名
    v.scene.includes(search) ||       // 场景
    v.filename.includes(search)       // 文件名
)
```

## 🔍 故障排查

### 问题 1：页面显示空白

**原因：** 目录列表无法读取

**解决：**
```bash
# 检查 Web 服务
ps aux | grep flask_voice

# 查看日志
tail -f logs/flask_voice.log

# 重启服务
pkill -f flask_voice
python3 flask_voice_server.py &
```

---

### 问题 2：音频无法播放

**原因：** 文件路径错误或 CORS 问题

**解决：**
```bash
# 检查文件是否存在
ls output/demo/*.mp3 | head -5

# 检查 Flask 代理
curl http://localhost:7091/api/health
```

---

### 问题 3：搜索无结果

**原因：** 文件名解析失败

**解决：**
```bash
# 检查文件名格式
ls output/demo/ | head -5
# 应该是：xiaoxiao_通用.mp3
```
