## 🚀 快速使用

### 1. 试听
```
http://localhost:7085/cute_voice.html
```

### 2. 选择语音
- **xiaoxiao** (温柔姐姐) - 最推荐
- **xiaoyi** (活泼妹妹) - 次推荐

### 3. 生成语音
```bash
cd /home/admin/projects/live-tts
bash start.sh

> s
输入文本：欢迎宝宝～来看直播呀！好想你哦～快坐下嘛～💕
语音 (默认 xiaoxiao): xiaoxiao
语速调整 (-50 到 +50): -10
文件名：welcome_cute.mp3
```

### 4. 批量生成
```python
# batch_cute.py
scripts = [
    ("welcome.mp3", "欢迎宝宝～来看直播呀！好想你哦～💕", "xiaoxiao", -10),
    ("thank.mp3", "谢谢宝宝～的关注！爱你哟～❤️", "xiaoxiao", -10),
    ("follow.mp3", "点点关注嘛～人家会想你的～✨", "xiaoyi", -5),
]
```
