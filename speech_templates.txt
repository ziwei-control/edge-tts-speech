# 直播常用口播文案

## 开场白

```
欢迎新进来的朋友们！
点点关注不迷路，主播带你上高速！
喜欢主播的记得点个关注哦！
```

## 感谢关注

```
感谢 XXX 的关注！
欢迎 XXX 加入直播间！
感谢 XXX 送出的礼物！
```

## 互动引导

```
觉得主播说得有道理的，扣个 1！
大家觉得怎么样？评论区告诉我！
想看的扣在公屏上！
```

## 产品/内容介绍

```
今天给大家带来的是...
这款产品有三个亮点...
让我详细给大家介绍一下...
```

## 促单/转化

```
限时优惠，只剩最后 10 单！
现在下单还送精美礼品！
错过今天就要等下次了！
```

## 下播预告

```
今天的直播就到这里了！
明天晚上 8 点，我们不见不散！
感谢大家的陪伴，晚安！
```

## 应急话术

```
网络有点卡，大家稍等一下！
让我调整一下设备！
这个问题我稍后回答！
```
