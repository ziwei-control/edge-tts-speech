系统会自动：

2. ✅ 保存为 MP3 文件
3. ✅ 在页面自动播放

## ⚠️ 常见问题

### Q1: 点击发送没反应？

**检查：**
1. 输入框是否有文字？
2. 浏览器控制台是否有错误？（F12）
3. 服务是否运行？

**解决：**
```bash
# 检查服务
curl http://localhost:7091/api/health

# 重启服务
cd /home/admin/projects/live-tts
bash start_voice_service.sh
```

---

### Q2: 语音不播放？

**可能原因：**
1. 浏览器静音
2. 音频文件未生成
3. 跨域问题

**检查：**
```bash
# 查看最新生成的音频
ls -lt /home/admin/projects/live-tts/output/api/ | head -5

# 测试播放
curl http://localhost:7086/audio/speech_xxx.mp3
```

---

### Q3: 语音生成失败？

**错误信息：**
```
生成失败：No audio was received
```

**原因：**
- 语音名称错误
- 网络连接问题
- Edge TTS 服务异常

**解决：**
```bash
# 测试 TTS API
curl -X POST http://localhost:7086/speak \
  -H "Content-Type: application/json" \
  -d '{"text":"测试","scene":"default"}'

# 查看日志
tail -f /home/admin/projects/live-tts/logs/api_server.log
```

---

### Q4: 中文乱码？

**表现：**
```
生成失败：???
```

**解决：**
- 确保输入是 UTF-8 编码
- 刷新页面（Ctrl + F5）
- 检查浏览器编码设置

## 🔗 API 端点

### 文字转语音 API

**请求：**
```http
POST /api/speak
Content-Type: application/json

{
  "text": "你好，测试",
  "scene": "default"
}
```

```json
{

  "filename": "speech_xxx.mp3",
```
