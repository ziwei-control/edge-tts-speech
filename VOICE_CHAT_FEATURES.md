## 🔧 技术实现

### 数据结构

**voice_data.json：**
```json
{
  "demo": [
    {"voiceName": "xiaoxiao", "scene": "通用"},
    {"voiceName": "xiaoyi", "scene": "通用"},
    ...
  ],
  "natural": [...],
  "cute": [...],
  "weak": [...]
}
```

---

### 加载逻辑

```javascript
// 1. 加载 JSON 数据
async function loadVoiceData() {
    const response = await fetch('voice_data.json');
    voiceData = await response.json();
    
    // 2. 提取唯一语音名
    const allVoices = new Set();
    Object.values(voiceData).forEach(voices => {
        voices.forEach(v => allVoices.add(v.voiceName));
    });
    
    // 3. 渲染选择器
    const voiceList = Array.from(allVoices).sort();
    renderVoiceSelector(voiceList);
}
```

---

### 切换逻辑

```javascript
// 点击语音名
function selectVoice(voice) {
    currentVoice = voice;
    document.getElementById('currentVoiceName').textContent = voice;
    
    // 更新选中状态
    document.querySelectorAll('.voice-option').forEach(opt => {
        opt.classList.toggle('active', opt.textContent === voice);
    });
    
    // 添加提示
    addAgentMessage(`已切换到 ${voice} 音色 ✨`);
}
```

---

### 发送语音

```javascript
// 发送文字时带上语音参数
async function sendText() {
    const response = await fetch(`${API_BASE}/speak`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            text: text,
            scene: currentScene,
            voice: currentVoice  // ⭐ 传递语音名
        })
    });
}
```

## 📈 性能优化

### 1. 懒加载

**默认只加载 6 个常用语音：**
```javascript
const commonVoices = ['xiaoxiao', 'xiaoyi', 'xiaoxuan', 'xiaomo', 'yunhao', 'yunye'];
renderVoiceSelector(commonVoices);
```

**点击展开才加载全部：**
```javascript
function toggleVoiceSelector() {
    if (voiceSelectorExpanded) {
        // 加载所有语音
        const allVoices = Array.from(allVoicesSet).sort();
        renderVoiceSelector(allVoices);
    }
}
```

---

### 2. 去重处理

**使用 Set 去重：**
```javascript
const allVoices = new Set();
Object.values(voiceData).forEach(voices => {
    voices.forEach(v => allVoices.add(v.voiceName));
});
```

**结果：** 152 个文件 → 20+ 个唯一语音名

---

### 3. 排序显示

**按字母排序：**
```javascript
const voiceList = Array.from(allVoices).sort();
// xiaochen, xiaocheng, xiaohan, ...
```

**方便查找：** 语音名按字母顺序排列
